Privacy Policy for YouTube Caption Summarizer

Introduction

This Privacy Policy describes how YouTube Caption Summarizer ("we", "our", or "the extension") collects, uses, and protects your information when you use our Chrome extension.

Information We Collect

The extension collects and processes:

  • YouTube video URLs you interact with
  • Video transcripts and captions
  • API keys you provide (OpenAI, Gemini, Groq)
  • User preferences and settings
  • Screenshots you capture within YouTube videos (stored locally only)

How We Use Information

We use the collected information to:

  • Generate video summaries
  • Process video captions
  • Save your preferences
  • Provide screenshot functionality
  • Improve extension functionality

Data Storage

  • API keys are stored securely in Chrome's sync storage
  • Screenshots and summaries are stored locally in your browser
  • No data is sent to our servers except when processing through AI services
  • All processing is done through secure API endpoints

Third-Party Services

We use:

  • OpenAI API
  • Google Gemini API
  • Groq API

These services have their own privacy policies and terms of service.

Data Security

  • All API communications are encrypted using HTTPS
  • API keys are encrypted in storage
  • No personal data is shared with third parties

User Rights

You can:

  • Delete stored API keys
  • Clear local storage
  • Disable the extension at any time

Changes to Privacy Policy

We may update this policy periodically. Changes will be posted in the Chrome Web Store listing.
